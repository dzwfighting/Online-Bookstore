##mongodb name: DuckBook

## Default Admin User
* username: admin1
* Password: admin1

## Installation & Set up

Install dependencies
```sh
npm install
```
Set up data
```sh
 npm run seed - install seed data
```
Turn on the local server and running on http://localhost:3000
```sh
npm start
```
